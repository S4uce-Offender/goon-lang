Current features:

**Simple mathematics:**

Exponentiation (**)

Unary Expressions (-, ~)

Addition (+)

Subtraction (-)

Multiplication (*)

Division (/)

**Boolean Comparisons**

**Numerical Comparison**

**String concatenation (using &lt;string&gt; + &lt;string&gt;)**



To build this project simply run make build.

USAGE

To use the REPL simply run the program.

To evaluate a file:
./gooner &lt;input file&gt;.goon
